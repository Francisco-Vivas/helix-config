# helix-config
My config files for [Helix](https://helix-editor.com/).

## Installation
### Fedora
```
sudo dnf copr enable varlad/helix
sudo dnf install helix
git clone git@github.com:Francisco-Vivas/helix-config.git ~/.config/helix --depth 1
```

### MacOs
```
brew install helix
&& git clone git@github.com:Francisco-Vivas/helix-config.git ~/.config/helix --depth 1
```

## Remove old versions
```
rm -rf ~/.config/helix
```
